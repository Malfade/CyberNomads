## Описание

Скачайте файл Windows_Logs01.evtx, найдите событие входа в систему и определите:
1. Имя учетной записи, выполнившей вход.
2. Тип входа (например, интерактивный, удалённый и т.д.).

Формат ответа: xxxxx; x
