# Sysmon Practies — материалы для практикума

Задания 18–21 выполняются **на учебной Windows-ВМ**, а не по скачанному архиву.

1. **Создание файла (id 18)** — создайте `obuchenie.bat` на рабочем столе; ответ — Event ID.
2. **Изменение файла (id 19)** — добавьте в bat строки `@echo off` и `start calc.exe`.
3. **Запуск процесса (id 20)** — запустите bat-файл.
4. **Удаление файла (id 21)** — удалите bat-файл.

Шаблон: `obuchenie.bat` в этой папке. Журнал Sysmon смотрите на ВМ курса.
